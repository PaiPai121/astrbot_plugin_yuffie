# Yuffie AstrBot 插件部署说明

## 📋 部署步骤

### 步骤 1：安装依赖

```bash
cd /path/to/astrbot_plugin_yuffie
pip3 install -r requirements.txt
```

### 步骤 2：配置 AstrBot

将插件文件夹放置到 AstrBot 插件目录：

```bash
# AstrBot 目录结构
astrbot/
├── plugins/
│   └── astrbot_plugin_yuffie/  ← 放这里
├── config/
└── ...
```

### 步骤 3：启动 AstrBot

```bash
# 启动 AstrBot
python3 -m astrbot

# 或者使用 Docker
docker run -d \
  -v /path/to/plugins:/app/plugins \
  -p 8501:8501 \
  astrbot/astrbot
```

### 步骤 4：验证部署

#### 4.1 检查插件状态

在 AstrBot 控制台查看日志：


## 🔧 配置说明

### config.yaml

```yaml
# 核心监控参数
monitor:
  symbol: "GC=F"                  # 监控标的
  cooldown_minutes: 30            # 预警冷却时间

# 量化预警阈值
atr_strategy:
  k_multiplier: 2.5               # ATR 触发倍数
  fallback_threshold: 10.0        # 降级绝对阈值

# 数据源
data_sources:
  primary_ws: "wss://stream.binance.com:9443/ws/paxgusdt@trade"
  domestic_gold_api: "http://push2.eastmoney.com/api/qt/stock/get?secid=118.Au9999&fields=f43,f60"
```

如果 8501 端口被占用，修改 `main.py`：

```python
_streamlit_process = subprocess.Popen(
```

## 🚨 常见问题

### Q1: Streamlit 启动失败

**症状：**
```
[Yuffie] ❌ Streamlit 启动失败：...
```

**解决：**
```bash
# 检查 Streamlit 是否安装
pip3 install streamlit

# 检查端口是否被占用
lsof -i :8501

# 修改端口或关闭占用进程
```

### Q2: 国内金价获取失败

**症状：**
```
[PriceFetcher] 东方财富 API 失败：...
```

**解决：**
- 检查网络连接
- 验证 API 是否可用
- 查看防火墙设置

### Q3: 指令无响应

**症状：**
- 发送 `/订阅` 等指令无回复

**解决：**
```bash
# 检查 AstrBot 日志
tail -f astrbot/logs/bot.log

# 验证插件是否加载
grep "Yuffie" astrbot/logs/bot.log
```
