# Yuffie AstrBot 插件 - 部署指南

## 🚀 快速部署

### 步骤 1：安装依赖

```bash
cd /path/to/astrbot_plugin_yuffie
pip3 install -r requirements.txt
```

### 步骤 2：放置到 AstrBot 插件目录

```bash
# 复制插件到 AstrBot 插件目录
cp -r astrbot_plugin_yuffie /path/to/astrbot/plugins/

# 目录结构
astrbot/
├── plugins/
│   └── astrbot_plugin_yuffie/  ← 放这里
│       ├── main.py
│       ├── web_app.py
│       ├── core/
│       └── plugins/
├── config/
└── ...
```

### 步骤 3：启动 AstrBot

```bash
# 方式 1：直接启动
cd /path/to/astrbot
python3 -m astrbot

# 方式 2：使用 Docker
docker run -d \
  -v /path/to/astrbot:/app \
  -v /path/to/astrbot_plugin_yuffie:/app/plugins/astrbot_plugin_yuffie \
  -p 8501:8501 \
  astrbot/astrbot
```

## ⚙️ 配置说明

### config.yaml

```yaml
# 核心监控参数
monitor:
  symbol: "GC=F"                  # 监控标的
  cooldown_minutes: 30            # 预警冷却时间

# 量化预警阈值
atr_strategy:
  k_multiplier: 2.5               # ATR 触发倍数
  fallback_threshold: 10.0        # 降级绝对阈值

# 数据源
data_sources:
  primary_ws: "wss://stream.binance.com:9443/ws/paxgusdt@trade"
  domestic_gold_api: "http://push2.eastmoney.com/api/qt/stock/get?secid=118.Au9999&fields=f43,f60"
```

### 端口配置

如果 8501 端口被占用，修改 `main.py`：

```python
# 在 start_plugin() 函数中修改端口
cmd = f"{sys.executable} -m streamlit run \"{web_app_path}\" --server.port 8502 ..."
```

## 🚨 常见问题

### Q1: Streamlit 启动失败

**症状：**
```
[Yuffie] ❌ Web 仪表盘启动失败：...
```

**解决：**
```bash
# 检查 Streamlit 是否安装
pip3 install streamlit

# 检查端口是否被占用
lsof -i :8501

# 修改端口或关闭占用进程
```

### Q2: 国内金价获取失败

**症状：**
```
[PriceFetcher] 东方财富 API 失败：...
```

**解决：**
- 检查网络连接
- 验证 API 是否可用
- 查看防火墙设置

### Q3: 指令无响应

**症状：**
- 发送 `/订阅` 等指令无回复

**解决：**
```bash
# 检查 AstrBot 日志
tail -f /path/to/astrbot/logs/bot.log

# 验证插件是否加载
grep "Yuffie" /path/to/astrbot/logs/bot.log
```

### Q4: Web 面板无法访问

**症状：**
- 浏览器无法打开 http://localhost:8501

**解决：**
```bash
# 检查 Streamlit 进程
ps aux | grep streamlit

# 查看 Streamlit 日志
tail -f /path/to/astrbot/logs/streamlit.log

# 检查防火墙
ufw allow 8501/tcp
```
